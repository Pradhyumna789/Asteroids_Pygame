# Using this project<br>
git clone https://github.com/Pradhyumna789/Asteroids_Pygame.git<br>
cd the directory<br>
run -> python3 main.py<br>

W -> move forward<br>
S -> move backward<br>
A -> move left<br>
D -> move right<br>
SPACE -> Fire bullets<br>
